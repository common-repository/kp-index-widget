Afficher l'index KP sur votre site web

== Description ==

Cette extension vous permet d'afficher, dans un Widget, les prévisions géomagnétique du jour en cours et des deux jours suivants, avec les données fournies par le centre de prédictions spaciales de l'administration nationale océanique et atmosphérique américaine.

L'index KP va de 1 à 9, et indique des perturbations à partir de 5.

== Installation ==

1. Uploadez les fichiers de l'extension dans le répertoire '/wp-content/plugins/kp-index-smt', ou installez le à travers le répertoire d'extension directement
2. Activez l'extension
3. Ajoutez le widget à l'un de vos blocs et paramêtrez le.
